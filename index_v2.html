Basic Problem 5 / 18

함수와 역함수의 교점 구조

\(y=x\) 위 교점 vs. \(y=x\) 밖 교점

Fixed Point on \(y=x\)
\((x_0,y_0)\) on \(y=f(x)\) & \(y=f^{-1}(x)\):
\(y_0 = x_0 \Rightarrow f(x_0) = x_0\)
2-cycle off \(y=x\)
\(y_0 \neq x_0 \Rightarrow (y_0, x_0)\) also intersection
(symmetric w.r.t. \(y=x\))
\(f(f(x_0)) = x_0, \quad f(x_0) \neq x_0\)
Application
\((x,y)\) satisfies \(y=a^x,\; x=a^y\)
\(\Rightarrow\; x = a^{a^x} = f(f(x))\)

∴ intersections \(\Leftrightarrow f^2(x)=x\)
Classification Rule
incr. \(f\) → off-diagonal: impossible
decr. \(f\) → off-diagonal: possible
Basic Problem 6 / 18

증가함수 (\(a > 1\)) 교점 분류

solution on \(y=x\)
\(a^x = x \;\Leftrightarrow\; \ln a = \dfrac{\ln x}{x}\)
\(h(x) = \dfrac{\ln x}{x},\quad h'(x) = \dfrac{1-\ln x}{x^2}\)
\(\displaystyle\max_x h = h(e) = \dfrac{1}{e}\)
\(\ln a \leq \dfrac{1}{e}\;\Leftrightarrow\; a \leq e^{1/e}\)
Graph: \(h(x) = \dfrac{\ln x}{x}\)
Range of \(a\) \(-\ln a\) (\(=\eta\)) Count Structure
\(1 < a < e^{1/e}\) \(-1/e < \eta < 0\) 2 2 on \(y=x\)
\(a = e^{1/e}\) \(\eta = -1/e\) 1 tangent pt. on \(y=x\)
\(a > e^{1/e}\) \(\eta < -1/e\) 0 none
Basic Problem 7 / 18

감소함수 (\(0 < a < 1\)) 교점 분류

2-cycle Condition
\(\eta := -\ln a > 0,\quad H_\eta(x) := \eta e^{-\eta e^{-x}} - x\)
\(H'_\eta(x) = \eta^2 e^{-x - \eta e^{-x}} - 1 \;\leq\; \dfrac{\eta}{e} - 1\)

\(\eta > e \Rightarrow H_\eta\) has local extrema → 3 roots
\(\eta \leq e \Rightarrow H_\eta\) decreasing → 1 root
\(\eta = -\ln a > e \;\Leftrightarrow\; a < e^{-e}\)
Root Structure of \(H_\eta\)
Range of \(a\) \(\eta = -\ln a\) Count Structure
\(0 < a < e^{-e}\) \(\eta > e\) 3 1 on \(y=x\) + 2-cycle pair
\(e^{-e} \leq a < 1\) \(0 < \eta \leq e\) 1 1 on \(y=x\)
Lambert W 8 / 18

Lambert \(W\) 함수

\(w \mapsto we^w\) 의 역함수

Definition
\[ W(xe^x) = x \]
Real Branches
  • \(W_0\): \(x \in [-e^{-1}, \infty)\), \(W_0 \geq -1\)
  • \(W_{-1}\): \(x \in [-e^{-1}, 0)\), \(W_{-1} \leq -1\)
  • \(x \in (-e^{-1}, 0)\): 2 real branches
  • \(W_0(-1/e) = W_{-1}(-1/e) = -1\)
\(x \in (-1/e,\; 0) \Rightarrow\) 2 real branches
\(x \in [0, \infty) \Rightarrow\) 1 real branch \((W_0)\)
\(x < -1/e \Rightarrow\) no real branches

Generalization 10 / 18

일반화된 함수 \(f(x) = ka^x + b\)

\(y = f(x)\) 와 \(y = f^{-1}(x)\) 의 교점

Setup
  • \(a > 0,\; a \neq 1,\; k \neq 0\)
  • \(b \in \mathbb{R}\) (vertical shift)
  • original: \(k = 1,\; b = 0\)
Key Example
Ex: \(f(x) = 4 \cdot (0.5)^x\)
intersections \((1,2)\), \((2,1)\) — integer pair!
\(\;4 \cdot 0.5^1 = 2, \quad 4 \cdot 0.5^2 = 1\)
fixed pt. on \(y=x\): \(x_* \approx 1.457\)
Intersection Condition
\(y = ka^x + b\) and \(x = ka^y + b\) simultaneously
\(\Leftrightarrow\; f(f(x)) = x\)

on \(y=x\): \(\; f(x) = x\)
off \(y=x\): \(\; f(x) \neq x,\; f(f(x)) = x\)
same structure — unified via single \(\eta\)
Generalization 11 / 18

Control Variable \(\eta\)

translation removal → Lambert \(W\) form

Substitution
\(X := x - b,\quad Y := y - b\)
\(K := ka^b,\quad c := \ln a\)

result:
\(Y = Ke^{cX},\quad X = Ke^{cY}\)
Definition of \(\eta\)
\[ \eta := -cK = -ka^b \ln a \]
fixed pt. \((X=Y)\):
\(Xe^{-cX} = K\;\Rightarrow\;(-cX)e^{-cX} = -cK = \eta\)
\(\Rightarrow\; -cX = W_j(\eta)\)
Generalized Formula
\(x_* = b - \dfrac{1}{\ln a}\,W_j(\eta)\)
original (\(k=1, b=0\)): \(\eta = -\ln a\), \(\; x_* = -W(-\ln a)/\ln a\) — consistent
Generalization 12 / 18

증가형 (\(k \ln a > 0\)) 교점 분류

Monotonicity Argument
\(f\) incr., \((x_0, y_0)\) intersection, \(x_0 < y_0\):
\(f(x_0) < f(y_0) \;\Rightarrow\; y_0 < x_0 \;\Rightarrow\!\Leftarrow\)

∴ incr. \(f\): off-diagonal intersections impossible
all intersections lie on \(y=x\)
\(k\ln a > 0 \;\Rightarrow\; \eta = -ka^b\ln a < 0\)
#intersections = #real branches of Lambert \(W\)
Classification
\(\eta\) range Count Note
\(-1/e < \eta < 0\) 2 \(W_0, W_{-1}\) branches
\(\eta = -1/e\) 1 tangent pt.
\(\eta < -1/e\) 0 none
stability: \(f'(x_*) = -W_j(\eta)\)
\(W_0\) branch: \(|W_0| < 1\) → attracting
\(W_{-1}\) branch: \(|W_{-1}| > 1\) → repelling
Generalization 13 / 18

감소형 (\(k \ln a < 0\)) 교점 분류

Critical Value
\(\eta = |k|a^b|\ln a| > 0\)
\(f'(x_*) = -W_0(\eta)\)

\(|f'(x_*)| = 1 \;\Leftrightarrow\; W_0(\eta) = 1\)
\(\Leftrightarrow\; \eta = 1 \cdot e^1 = e\)
\(\eta > e \Rightarrow\) fixed point unstable
\(\Rightarrow\) 2-cycle pair off \(y=x\) emerges
Classification
\(\eta\) range Count Structure
\(0 < \eta \leq e\) 1 Fixed pt. on \(y=x\)
\(\eta > e\) 3 1 on \(y=x\) + 2-cycle pair
Ex. \(f(x) = 4 \cdot (0.5)^x\)
\(\eta = 4\ln 2 \approx 2.773 > e \approx 2.718\)
∴ \(\eta > e\) → 3 intersections
\(\quad(1,\;2),\quad(1.457\ldots,\;1.457\ldots),\quad(2,\;1)\)

Infinite Tower 15 / 18

무한 지수탑과 Lambert \(W\) 함수

Limit Equation
\(T = a^T\) (self-similar)
\(\Rightarrow -T\ln a = W(-\ln a)\)

\(\equiv\) fixed-point condition \(x_* = a^{x_*}\)
\[ T = -\frac{W(-\ln a)}{\ln a} \]
Convergence Range
\[ e^{-e} \leq a \leq e^{1/e} \]
\(e^{-e} \approx 0.0660,\quad e^{1/e} \approx 1.4447\)
convergence \(\Leftrightarrow\) stability: \(|f'(T)| = |T\ln a| < 1\)
\(\Leftrightarrow |W(-\ln a)| < 1\)
\(x_{n+1} = a^{x_n},\quad x_0 \in [e^{-e},\, e^{1/e}] \;\Rightarrow\; x_n \to T\)
