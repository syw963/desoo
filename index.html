함수와 역함수의 교점 구조

\(y=x\) 위 교점 vs. \(y=x\) 밖 교점

Fixed Point on \(y=x\)
\((x_0,y_0)\) on \(y=f(x)\) & \(y=f^{-1}(x)\):
\(y_0 = x_0 \Rightarrow f(x_0) = x_0\)
2-cycle off \(y=x\)
\(y_0 \neq x_0 \Rightarrow (y_0, x_0)\) also intersection
(symmetric w.r.t. \(y=x\))
\(f(f(x_0)) = x_0, \quad f(x_0) \neq x_0\)
Application
\((x,y)\) satisfies \(y=a^x,\; x=a^y\)
\(\Rightarrow\; x = a^{a^x} = f(f(x))\)

∴ intersections \(\Leftrightarrow f^2(x)=x\)
Classification Rule
incr. \(f\) → off \(y=x\): impossible
decr. \(f\) → off \(y=x\): possible

\(a^x = x\) — Lambert \(W\) 형태로 정리

fixed-pt. derivation

\(a^x = x \;\Rightarrow\; e^{-x\eta} = x \quad (\eta = -\ln a)\)
\(\Rightarrow\; 1 = x e^{x\eta} \;\Rightarrow\; x\eta e^{x\eta} = \eta\)
\(\Rightarrow\; x\eta = W(\eta) \;\Rightarrow\; \boxed{\,x^* = \dfrac{W(\eta)}{\eta}\,}\)
Lambert \(W\) recall
\(W(xe^x) = x\)
increasing \(\;(a > 1,\;\eta < 0)\)
\(\eta \in (-1/e,\;0)\): \(W_0,\;W_{-1}\) real → 2 fixed pts.
\(\eta = -1/e\;\;(a = e^{1/e})\): \(W_0 = W_{-1} = -1\) → 1
\(\eta < -1/e\;\;(a > e^{1/e})\): no real \(W\) → 0
decreasing \(\;(0 < a < 1,\;\eta > 0)\)
\(W_0(\eta) > 0\) always → 1 fixed pt. always
2-cycle analysis → next slide

감소형 — \(h(x)=0\) 변환

\(x = a^{a^x}\;\) via \(\;\eta = -\ln a > 0\)

Substitution
\(\begin{cases} y = a^x \\ x = a^y \end{cases} \Rightarrow x = a^{a^x}\)
\(\Rightarrow\; \ln x = a^x \cdot \ln a\)
\(\phantom{\Rightarrow\; \ln x\,} = a^x \cdot (-\eta) \quad (\because \ln a = -\eta)\)
\(\phantom{\Rightarrow\; \ln x\,} = -\eta e^{-\eta x} \quad (\because a^x = e^{-\eta x})\)
\(h(x) = \ln x + \eta e^{-\eta x}\)
zeros of \(h\) \(\;\Leftrightarrow\;\) solutions of \(x = a^{a^x}\)
Critical pts. of \(h\)
\(h'(x) = \dfrac{1}{x} - \eta^2 e^{-\eta x} = 0\)
\(\Rightarrow\; (-\eta x)\,e^{-\eta x} = -\dfrac{1}{\eta}\)
\(\Rightarrow\; \boxed{x = -\dfrac{1}{\eta}\,W\!\left(-\dfrac{1}{\eta}\right)}\)
real \(W\) exists \(\;\Leftrightarrow\; {-\dfrac{1}{\eta}} \geq {-\dfrac{1}{e}} \;\Leftrightarrow\; \eta \geq e\)

극대 \(> 0\), 극소 \(< 0\) 증명

proof for \(\eta > e\)

Setting up \(F(t)\)
\(t = \eta x \;\Rightarrow\) critical pt. cond.: \(\;te^{-t} = \dfrac{1}{\eta}\)
\(\eta > e \;\Rightarrow\; \dfrac{1}{\eta} < \dfrac{1}{e} = \max(te^{-t})\)
\(\Rightarrow\; 0 < t_1 < 1 < t_2\)
at \(x = t/\eta\): \(\;\eta e^{-t} = \dfrac{1}{t},\quad \ln\eta = t - \ln t\)
\(h\!\left(\tfrac{t}{\eta}\right) = \ln t - \ln\eta + \tfrac{1}{t} = 2\ln t - t + \tfrac{1}{t}\)
\(F(t) \;=\; 2\ln t - t + \dfrac{1}{t}\)
Sign of \(F(t)\)
\(F'(t) = \dfrac{2}{t} - 1 - \dfrac{1}{t^2} = -\dfrac{(t-1)^2}{t^2} \leq 0\)
\(F\) strictly decr.,\quad \(F(1) = 2\cdot 0 - 1 + 1 = 0\)
\(t_1 < 1 \;\Rightarrow\; F(t_1) > 0\quad\) (local max \(> 0\))
\(t_2 > 1 \;\Rightarrow\; F(t_2) < 0\quad\) (local min \(< 0\))
\(h(x_1) = F(t_1) > 0,\quad h(x_2) = F(t_2) < 0\)
\(\Rightarrow\; h\) crosses \(x\)-axis 3 times

무한 지수탑과 Lambert \(W\) 함수

Limit Equation
\(T = a^T\) (self-similar)
\(\Rightarrow -T\ln a = W(-\ln a)\)

\(\equiv\) fixed-point condition \(x^* = a^{x^*}\)
\[ T = -\frac{W(-\ln a)}{\ln a} \]
수렴 범위
\[ e^{-e} \leq a \leq e^{1/e} \]
\(e^{-e} \approx 0.0660,\quad e^{1/e} \approx 1.4447\)
convergence \(\Leftrightarrow\) stability: \(|f'(T)| = |T\ln a| < 1\)
\(\Leftrightarrow |W(-\ln a)| < 1\)
\(x_{n+1} = a^{x_n},\quad a \in [e^{-e},\, e^{1/e}] \;\Rightarrow\; x_n \to T\)
