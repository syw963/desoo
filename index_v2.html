함수와 역함수의 교점 구조

\(y=x\) 위 교점 vs. \(y=x\) 밖 교점

Fixed Point on \(y=x\)
\((x_0,y_0)\) on \(y=f(x)\) & \(y=f^{-1}(x)\):
\(y_0 = x_0 \Rightarrow f(x_0) = x_0\)
2-cycle off \(y=x\)
\(y_0 \neq x_0 \Rightarrow (y_0, x_0)\) also intersection

\(f(f(x_0)) = x_0, \quad f(x_0) \neq x_0\)

\((x,y)\) satisfies \(y=a^x,\; x=a^y\)

\(a^x = x\) — Lambert \(W\) 형태로 정리

fixed-pt. derivation

\(a^x = x \;\Rightarrow\; e^{-x\eta} = x \quad (\eta = -\ln a)\)
\(\xRightarrow{\;u\,=\,x\eta\;}\; ue^u = \eta \;\Rightarrow\; \boxed{\,x_* = \dfrac{W(\eta)}{\eta}\,}\)
increasing \(\;(a > 1,\;\eta < 0)\)
\(\eta \in (-1/e,\;0)\): \(W_0,\;W_{-1}\) real → 2 fixed pts.
\(\eta = -1/e\;\;(a = e^{1/e})\): \(W_0 = W_{-1} = -1\) → 1
\(\eta < -1/e\;\;(a > e^{1/e})\): no real \(W\) → 0
decreasing \(\;(0 < a < 1,\;\eta > 0)\)
\(W_0(\eta) > 0\) always → 1 fixed pt. always
\(|f'(x_*)| = W_0(\eta)\)
\(W_0(\eta) > 1 \;\Leftrightarrow\; \eta > e \;\Leftrightarrow\; a < e^{-e}\) → 2-cycle

감소형 — \(h(x)=0\) 변환

\(x = a^{a^x}\;\) via \(\;\eta = -\ln a > 0\)

Substitution
\(a = e^{-\eta} \;\Rightarrow\; a^x = e^{-\eta x}\)
\(x = a^{a^x} \;\Rightarrow\; \ln x = a^x \ln a\)
\(\phantom{\ln x\,} = -\eta e^{-\eta x}\)
\(h(x) = \ln x + \eta e^{-\eta x}\)
zeros of \(h\) \(\;\Leftrightarrow\;\) solutions of \(x = a^{a^x}\)
Critical pts. of \(h\)
\(h'(x) = \dfrac{1}{x} - \eta^2 e^{-\eta x} = 0\)
\(\Rightarrow\; (-\eta x)\,e^{-\eta x} = -\dfrac{1}{\eta}\)
\(\Rightarrow\; \boxed{x = -\dfrac{1}{\eta}\,W\!\left(-\dfrac{1}{\eta}\right)}\)
real \(W\) exists \(\;\Leftrightarrow\; {-\dfrac{1}{\eta}} \geq {-\dfrac{1}{e}} \;\Leftrightarrow\; \eta \geq e\)

극대 \(> 0\), 극소 \(< 0\) 증명

proof for \(\eta > e\)

Setting up \(F(t)\)
\(t = \eta x \;\Rightarrow\) critical pt. cond.: \(\;te^{-t} = \dfrac{1}{\eta}\)
\(\eta > e \;\Rightarrow\; \dfrac{1}{\eta} < \dfrac{1}{e} = \max(te^{-t})\)
\(\Rightarrow\; 0 < t_1 < 1 < t_2\)
at \(x = t/\eta\): \(\;\eta e^{-t} = \dfrac{1}{t},\quad \ln\eta = t - \ln t\)
\(h\!\left(\tfrac{t}{\eta}\right) = \ln t - \ln\eta + \tfrac{1}{t} = 2\ln t - t + \tfrac{1}{t}\)
\(F(t) \;=\; 2\ln t - t + \dfrac{1}{t}\)
Sign of \(F(t)\)
\(F'(t) = \dfrac{2}{t} - 1 - \dfrac{1}{t^2} = -\dfrac{(t-1)^2}{t^2} \leq 0\)
\(F\) strictly decr.,\quad \(F(1) = 2\cdot 0 - 1 + 1 = 0\)
\(t_1 < 1 \;\Rightarrow\; F(t_1) > 0\quad\) (local max \(> 0\))
\(t_2 > 1 \;\Rightarrow\; F(t_2) < 0\quad\) (local min \(< 0\))
\(h(x_1) = F(t_1) > 0,\quad h(x_2) = F(t_2) < 0\)
\(\Rightarrow\; h\) crosses \(x\)-axis 3 times

일반화된 함수 \(f(x) = ka^x + b\)

\(y = f(x)\) 와 \(y = f^{-1}(x)\) 의 교점

Setup
  • \(a > 0,\; a \neq 1,\; k \neq 0\)
  • \(b \in \mathbb{R}\) (vertical shift)
  • original: \(k = 1,\; b = 0\)
Key Example
Ex: \(f(x) = 4 \cdot (0.5)^x\)
intersections \((1,2)\), \((2,1)\) — integer pair!
\(\;4 \cdot 0.5^1 = 2, \quad 4 \cdot 0.5^2 = 1\)
fixed pt. on \(y=x\): \(x_* \approx 1.457\)
Intersection Condition
\(y = ka^x + b\) and \(x = ka^y + b\) simultaneously
\(\Leftrightarrow\; f(f(x)) = x\)

on \(y=x\): \(\; f(x) = x\)
off \(y=x\): \(\; f(x) \neq x,\; f(f(x)) = x\)
same structure — unified via single \(\eta\)
Generalization 15 / 22

Control Variable \(\eta\)

translation removal → Lambert \(W\) form

Substitution
\(X := x - b,\quad Y := y - b\)
\(K := ka^b,\quad c := \ln a\)

result:
\(Y = Ke^{cX},\quad X = Ke^{cY}\)
Definition of \(\eta\)
\[ \eta := -cK = -ka^b \ln a \]
fixed pt. \((X=Y)\):
\(Xe^{-cX} = K\;\Rightarrow\;(-cX)e^{-cX} = -cK = \eta\)
\(\Rightarrow\; -cX = W_j(\eta)\)
Generalized Formula
\(x_* = b - \dfrac{1}{\ln a}\,W_j(\eta)\)
original (\(k=1, b=0\)): \(\eta = -\ln a\), \(\; x_* = -W(-\ln a)/\ln a\) — consistent

Monotonicity Argument
\(f\) incr., \((x_0, y_0)\) intersection, \(x_0 < y_0\):
\(f(x_0) < f(y_0) \;\Rightarrow\; y_0 < x_0 \;\Rightarrow\!\Leftarrow\)

∴ incr. \(f\): off-diagonal intersections impossible
all intersections lie on \(y=x\)
\(k\ln a > 0 \;\Rightarrow\; \eta = -ka^b\ln a < 0\)
#intersections = #real branches of Lambert \(W\)

\(\eta < -1/e\) 0 none

무한 지수탑과 Lambert \(W\) 함수

Limit Equation
\(T = a^T\) (self-similar)
\(\Rightarrow -T\ln a = W(-\ln a)\)

\(\equiv\) fixed-point condition \(x_* = a^{x_*}\)
\[ T = -\frac{W(-\ln a)}{\ln a} \]
Convergence Range
\[ e^{-e} \leq a \leq e^{1/e} \]
\(e^{-e} \approx 0.0660,\quad e^{1/e} \approx 1.4447\)
convergence \(\Leftrightarrow\) stability: \(|f'(T)| = |T\ln a| < 1\)
\(\Leftrightarrow |W(-\ln a)| < 1\)
\(x_{n+1} = a^{x_n},\quad a \in [e^{-e},\, e^{1/e}] \;\Rightarrow\; x_n \to T\)
