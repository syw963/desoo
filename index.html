함수와 역함수의 교점 구조

\(y=x\) 위 교점 vs. \(y=x\) 밖 교점

\(y=x\) 위 고정점
\(y = f(x)\) 와 \(y = f^{-1}(x)\) 의 교점 \((x_0,y_0)\) 에서
\(y_0 = x_0\) 이면 \(\;f(x_0) = x_0\)
\(y=x\) 밖 2주기점
\(y_0 \neq x_0\) 이면 \((y_0, x_0)\) 도 교점
\((y=x\) 에 대해 대칭\()\)
\(f(f(x_0)) = x_0, \quad f(x_0) \neq x_0\)
원래 문제에 적용
교점 \((x,y)\) 가 \(y=a^x\), \(x=a^y\) 를 만족
\(\Rightarrow\; x = a^{a^x} = f(f(x))\)

따라서 교점은 \(f^2(x)=x\) 의 해
분류 원칙
증가형 \(f\) → \(y=x\) 밖 교점 불가
감소형 \(f\) → \(y=x\) 밖 교점 가능

증가함수 (\(a > 1\)) 교점 분류

\(y=x\) 위 해의 조건
\(a^x = x \;\Leftrightarrow\; \ln a = \dfrac{\ln x}{x}\)
\(h(x) = \dfrac{\ln x}{x},\quad h'(x) = \dfrac{1-\ln x}{x^2}\)
최댓값: \(\;h(e) = \dfrac{1}{e}\)
\(\ln a \leq \dfrac{1}{e}\;\Leftrightarrow\; a \leq e^{1/e}\) 일 때 해 존재
그래프 \(h(x) = \dfrac{\ln x}{x}\)
\(a\) 의 범위 \(-\ln a\) (\(=\eta\)) 교점 수 구조
\(1 < a < e^{1/e}\) \(-1/e < \eta < 0\) 2 \(y=x\) 위 2개
\(a = e^{1/e}\) \(\eta = -1/e\) 1 \(y=x\) 위 접점
\(a > e^{1/e}\) \(\eta < -1/e\) 0 교점 없음
기본 문제 7 / 18

감소함수 (\(0 < a < 1\)) 교점 분류

2주기점 존재 조건
\(\eta := -\ln a > 0,\quad H_\eta(x) := \eta e^{-\eta e^{-x}} - x\)
\(H'_\eta(x) = \eta^2 e^{-x - \eta e^{-x}} - 1 \;\leq\; \dfrac{\eta}{e} - 1\)

\(\eta > e\) 이면 \(H_\eta\) 가 극값을 가짐 → 근 3개
\(\eta \leq e\) 이면 \(H_\eta\) 감소 → 근 1개
\(\eta = -\ln a > e \;\Leftrightarrow\; a < e^{-e}\)
\(H_\eta\) 의 근 구조 변화
\(a\) 의 범위 \(\eta = -\ln a\) 교점 수 구조
\(0 < a < e^{-e}\) \(\eta > e\) 3 \(y=x\) 위 1개 + 대칭쌍 2개
\(e^{-e} \leq a < 1\) \(0 < \eta \leq e\) 1 \(y=x\) 위 1개

Lambert \(W\) 함수

\(w \mapsto we^w\) 의 역함수

정의
\[ W(xe^x) = x \]
실수 가지
  • \(W_0\): \(x \in [-e^{-1}, \infty)\), \(W_0 \geq -1\)
  • \(W_{-1}\): \(x \in [-e^{-1}, 0)\), \(W_{-1} \leq -1\)
  • \(x \in (-e^{-1}, 0)\) 에서 두 실수 가지
  • \(W_0(-1/e) = W_{-1}(-1/e) = -1\)
\(x \in (-1/e,\; 0)\)이면 실수 가지 2개
\(x \in [0, \infty)\)이면 실수 가지 1개 (\(W_0\))
\(x < -1/e\)이면 실수 가지 없음

\(y=a^x,\; x=a^y\) — Lambert \(W\)로 완전 분류

\(y=x\) 위의 해: \(\quad x_* = \dfrac{W(\eta)}{\eta}\) \(\quad\) (\(\eta = -\ln a\))

\(a = 1\) \(\eta = 0\) 1

\(a = e^{1/e}\) \(\eta = -1/e\) 1
\(a > e^{1/e}\) \(\eta < -1/e\) 0
\(a=\) 0.040

임계값 \(e^{-e} \approx 0.0660\), \(\;e^{1/e} \approx 1.4447\) — 두 경계는 \(|f'(x_*)| = 1\) 조건에서 유도

일반화된 함수 \(f(x) = ka^x + b\)

\(y = f(x)\) 와 \(y = f^{-1}(x)\) 의 교점

일반화 조건
  • \(a > 0,\; a \neq 1,\; k \neq 0\)
  • \(b\) 는 임의의 실수 (수직 이동)
  • 원래 문제: \(k = 1,\; b = 0\)
핵심 예시
예시: \(f(x) = 4 \cdot (0.5)^x\)
교점 \((1,2)\), \((2,1)\) — 정수쌍!
\(\;4 \cdot 0.5^1 = 2, \quad 4 \cdot 0.5^2 = 1\)
\(y=x\) 위 고정점: \(x_* \approx 1.457\)
교점 조건
\(y = ka^x + b\) 와 \(x = ka^y + b\) 동시 만족
\(\Leftrightarrow\; f(f(x)) = x\)

\(y=x\) 위: \(\; f(x) = x\)
\(y=x\) 밖: \(\; f(x) \neq x,\; f(f(x)) = x\)
원래 문제와 동일한 구조 — 하나의 매개변수 \(\eta\) 로 통합 가능
일반화 11 / 18

제어변수 \(\eta\) 도입

평행이동 제거 → Lambert \(W\) 형 변형

변수 치환
\(X := x - b,\quad Y := y - b\)
\(K := ka^b,\quad c := \ln a\)

변환 결과:
\(Y = Ke^{cX},\quad X = Ke^{cY}\)
제어변수 정의
\[ \eta := -cK = -ka^b \ln a \]
\(y=x\) 위의 해 \((X = Y)\):
\(Xe^{-cX} = K\;\Rightarrow\;(-cX)e^{-cX} = -cK = \eta\)
\(\Rightarrow\; -cX = W_j(\eta)\)
일반화된 교점 공식
\(x_* = b - \dfrac{1}{\ln a}\,W_j(\eta)\)
원래 문제 (\(k=1, b=0\)): \(\eta = -\ln a\), \(\; x_* = -W(-\ln a)/\ln a\) — 앞 공식과 일치

증가형 (\(k \ln a > 0\)) 교점 분류

단조성 논법
\(f\) 증가, \((x_0, y_0)\) 가 교점이고 \(x_0 < y_0\) 이면:
\(f(x_0) < f(y_0) \;\Rightarrow\; y_0 < x_0\) — 모순

∴ 증가형에서 \(y=x\) 밖 교점 불가능
모든 교점은 \(y=x\) 위에만 존재
\(k\ln a > 0 \;\Rightarrow\; \eta = -ka^b\ln a < 0\)
교점 수 = Lambert \(W\) 의 실수 가지 수
분류표
\(\eta\) 범위 교점 수 비고
\(-1/e < \eta < 0\) 2 \(W_0, W_{-1}\) 두 가지
\(\eta = -1/e\) 1 접점
\(\eta < -1/e\) 0 교점 없음
안정성: \(f'(x_*) = -W_j(\eta)\)
\(W_0\) 가지: \(|W_0| < 1\) → attracting
\(W_{-1}\) 가지: \(|W_{-1}| > 1\) → repelling

감소형 (\(k \ln a < 0\)) 교점 분류

임계값 유도
\(\eta = |k|a^b|\ln a| > 0\)
\(f'(x_*) = -W_0(\eta)\)

\(|f'(x_*)| = 1 \;\Leftrightarrow\; W_0(\eta) = 1\)
\(\Leftrightarrow\; \eta = 1 \cdot e^1 = e\)
\(\eta > e\) 이면 고정점이 불안정해짐
→ \(y=x\) 밖 2주기점 2개 추가
분류표
\(\eta\) 범위 교점 수 구조
\(0 < \eta \leq e\) 1 \(y=x\) 위 고정점
\(\eta > e\) 3 \(y=x\) 위 1 + 밖 2주기쌍
예시 \(f(x) = 4 \cdot (0.5)^x\)
\(\eta = 4\ln 2 \approx 2.773 > e \approx 2.718\)
∴ \(\eta > e\) → 교점 3개
\(\quad(1,\;2),\quad(1.457\ldots,\;1.457\ldots),\quad(2,\;1)\)

극한값의 방정식
\(T = a^T\) (위쪽 꼬리도 \(T\))
\(\Rightarrow -T\ln a = W(-\ln a)\)

이 방정식은 앞서 구한
고정점 조건 \(x_* = a^{x_*}\) 와 동일
\[ T = -\frac{W(-\ln a)}{\ln a} \]
수렴 범위
\[ e^{-e} \leq a \leq e^{1/e} \]
\(e^{-e} \approx 0.0660,\quad e^{1/e} \approx 1.4447\)
수렴 조건: 고정점의 안정성 \(|f'(T)| = |T\ln a| < 1\)
\(\Leftrightarrow |W(-\ln a)| < 1\) — Lambert \(W\) 로 직접 표현
